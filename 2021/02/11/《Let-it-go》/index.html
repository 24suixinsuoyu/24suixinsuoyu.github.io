《Let it go》

发表于 2021-02-11 更新于 2021-02-16 分类于 《Melodious Music》
本文字数： 1.7k 阅读时长 ≈ 2 分钟

姚贝娜 · 1981年9月26日－2015年1月16日 · 湖北武汉人 · 中国女歌手

Intro

随着动画电影《冰雪奇缘》一同火起来的主题歌《Let It Go》为这部影片贡献了最豪华绚丽的一幕冰上圆舞曲，身负冰封魔咒的女主角冰雪女王艾莎离开故乡后，释放魔力在风中载歌载舞，凭空舞出了一座水晶般耀眼夺目的冰雪城堡。电影中精彩呈现了外形和魔法的转变，而主题歌却在叙事，关于一个人心灵转变后突然具有的强大精神力。
